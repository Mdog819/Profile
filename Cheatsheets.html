If / Else statements The if/else statement executes a statement if a specified condition is true. If the condition is false, another statement in the optional else clause will be executed.
Declare and assign variables A variable is always used to store information. for example,strings and intergers. A variable can be declared using the keyword var followed by the name of the variable and type. An = sign is then followed to assign the variable a value.
loops A loop is used to do something repeatedly without having to write the code over and over again.Loops are handy if you want to run the same code over and over again, each time with a different value.
Functions A function is a block of statements that can be used repeatedly in a program.
